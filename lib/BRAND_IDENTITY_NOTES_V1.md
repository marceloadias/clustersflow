# BRAND_IDENTITY_NOTES_V1 — Identidade Visual (Clusters Flow)

## 1) Marca
A marca do sistema é o **Cluster** (assistente virtual): um cubo 3D minimalista.

## 2) Regras
- Templates são propriedade do sistema; dados do usuário não são compartilhados.
- O Cluster é “vivo” e aparece em todo o sistema (pode pausar, não excluir).
- Quando pausado, pode exibir badge numérico para dicas importantes.

## 3) Logo/mascote (rascunho)
- cubo 3D simples
- sem bordas aparentes (ideal final)
- dois pontos (olhos)
- estrela amarela como assinatura (sugestão de cor observada: #FFB400)

## 4) Direção de paleta
- Azul como base (marca)
- testar variações com verde/teal como acento secundário
- não seguir “manada”; visual simples e objetivo
- dark como padrão; light opcional (tokenizado)

## 5) Assinaturas de UI (o que dá identidade além da cor)
- Persiana (Shade) da biblioteca
- Régua de cores horizontal na barra inferior
- Mini-mapa da LP à direita
- Layers/camadas
- Cluster como selo (Gold, notificações, dicas)
